# Recommended labels

| Label | Purpose |
|---|---|
| `good first issue` | Small, documented, low-risk contribution |
| `help wanted` | Maintainers welcome community ownership |
| `adapter` | Vendor, protocol, sensor, or integration adapter |
| `architecture` | Contract or architectural concern |
| `hmi` | Avalonia operator UI |
| `digital-twin` | Browser Three.js viewer |
| `runtime` | Machine-host lifecycle and execution |
| `motion` | Axis simulation and control |
| `reliability` | Recovery, soak, performance, resilience |
| `security` | Security improvement, not vulnerability disclosure |
| `platform/windows` | Windows-specific behavior |
| `platform/linux` | Linux-specific behavior |
| `platform/macos` | macOS-specific behavior |
| `needs-triage` | Awaiting maintainer classification |
| `breaking-change` | Requires migration and major-version consideration |
